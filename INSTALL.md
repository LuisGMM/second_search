

# Installation guide

## Python version

This project uses python 3.11. Create a venv with `virtualenv`:
```bash
virtualenv -p python3.11 venv
```

## Dependencies

`lxml` dependencies:

```bash
sudo apt install libxml2-dev libxslt-dev python3-dev -y &&
sudo apt install python3-lxml -y
```
